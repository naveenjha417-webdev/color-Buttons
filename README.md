# Color Buttons

An interactive web application built using HTML, CSS, and JavaScript that allows users to change colors dynamically by clicking different buttons.

## Features

* Dynamic color switching
* Clean and responsive user interface
* Beginner-friendly JavaScript project
* Lightweight and fast performance

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Purpose

This project was created to practice DOM manipulation, event handling, and basic JavaScript concepts while building an interactive user interface.
